# 👨🏻‍💻 Davi Azevedo

**`Engenheiro de Software`**

Me chamo Davi Azevedo, sou formado em Análise e Desenvolvimento de Sistemas e atualmente curso Engenharia de Software. Sou apaixonado por tecnologia, desenvolvimento de software e criação de soluções modernas voltadas para web e mobile.

Tenho experiência no desenvolvimento de aplicações FullStack utilizando tecnologias como React, React Native, Node.js, NestJS, TypeScript e PostgreSQL. Também possuo conhecimento em APIs REST, bancos de dados, autenticação, deploy de aplicações e integração de sistemas.

Atualmente desenvolvo projetos próprios e soluções reais voltadas para sistemas web, aplicativos mobile e plataformas SaaS, sempre buscando evoluir minhas habilidades e criar aplicações modernas, performáticas e escaláveis.
